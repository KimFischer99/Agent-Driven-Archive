# Controlled Tags

Store controlled tag vocabularies and authority guidance here.

Actual tags are written into domain cleaned-data frontmatter.

## Usage

1. Define allowed tags per domain in this directory.
2. Agents and editors reference these lists when tagging content.
3. Review and update the vocabulary as the archive grows.

Keep one file per tag category or domain (e.g. `archive_tags.md`, `blog_tags.md`).
