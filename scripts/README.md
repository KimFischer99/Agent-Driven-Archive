# Scripts

This folder contains reusable starter scripts extracted from the source project.

## Categories

- workspace validation
- content manifest export
- search index generation
- OCR helpers
- PDF rendering helpers

## Design Rule

Only keep scripts here if they are:

- topic-agnostic
- path-agnostic or configurable
- safe to publish
- useful across multiple archive projects

Corpus-specific one-off repair scripts should stay out until they are generalized.
